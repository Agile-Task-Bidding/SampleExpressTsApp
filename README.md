# Sample Express TypeScript App

Experimenting with Express and TypeScript

Install dependencies:
```bash
npm init
```
Add an index.html file to `/build/public/`. A sample is provided here:
```html
<!DOCTYPE html>
<html>
  <body>
    <h1>Yeet</h1>
    <p>Cool</p>
  </body>
</html>
```
Compile and run:
```bash
npm run build
npm start
```
